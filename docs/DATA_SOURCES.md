# Data Sources

## Public / Aggregate Data
This project is designed to use public, aggregate datasets.

### Eviction filings
- Court-issued county-level eviction filing data (where available)
- Eviction Lab county-level eviction filings (public distribution)

### Optional proxy indicators
- Permit counts (electrical/building) from county or municipal open data portals

## Privacy and Ethics
This repository does not store tenant-level data such as leases, payment records, or personal identifiers.
Sensitive documents should remain offline or shared only through secure channels when required.
