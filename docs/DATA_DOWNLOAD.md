# Data Download Instructions

This project uses only public, aggregated datasets.

## Eviction Lab (County Court-Issued)
Source: https://data-downloads.evictionlab.org/#data-for-analysis/

Recommended file:
- county_court_issued_2000_2018.csv

Download and place locally (not committed to GitHub) under:
- data/raw/eviction_lab/

## Local Public Maintenance Proxies (Optional)
Examples:
- County permit data (electrical/building)
- HUD Fair Market Rents (FMR)

Place under:
- data/raw/local/
