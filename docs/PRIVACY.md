# Privacy & Ethical Use

This repository is designed for public, aggregate, county-level analysis.

## What is NOT included
- No tenant-level data
- No names, addresses, leases, payment histories, or case narratives
- No personally identifiable information (PII)

## What IS included
- Public, aggregate datasets (county-level)
- Reproducible code to compute an "eviction pressure" score
- Documentation sufficient for replication

## Intended use
This project is an early warning and oversight tool to support housing stability efforts, responsible journalism, and evidence-informed regulation. It is not intended for individual-level targeting or enforcement against specific tenants.

## Data access statement
Additional documents and supporting materials may be available upon request for legitimate oversight purposes, but are not stored in this repository.
