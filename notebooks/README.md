# Notebooks

## Notebook index
- **01_pitt_county_eviction_early_warning.ipynb**
  - Builds the Pitt pilot score using public county-level indicators.
- **02_bay_county_disaster_segmentation.ipynb**
  - Adds disaster-period segmentation for Bay County (Hurricane Michael context).

## How to run locally
1. Create a virtual environment
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Follow `data/README.md` to fetch public datasets
4. Run notebooks in order
